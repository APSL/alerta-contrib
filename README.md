Alerta Contrib
==============

Useful but non-essential additions to the alerta monitoring system
which are grouped together as "integrations" or "plugins".

Integrations
------------

  * Cloudwatch
  * Consul
  * Mailer
  * OpsWeekly
  * PagerDuty
  * Pinger
  * SNMPTrap
  * Amazon SQS
  * Supervisor
  * Syslog
  * URLmon

Plugins
-------

  * AMQP
  * Enhance
  * GeoIP
  * HipChat
  * InfluxDB
  * Logstash
  * Normalise
  * PagerDuty
  * Prometheus
  * Pushover.net
  * Slack
  * Amazon SNS
  * Syslog
  * Twilio SMS
