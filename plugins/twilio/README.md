Twilio SMS Plugin
=================

Send SMS messages for new alerts.

Installation
------------

    $ python setup.py install


Configuration
-------------

```
TWILIO_ACCOUNT_SID = 'ACxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'
TWILIO_AUTH_TOKEN = ''

TWILIO_TO_NUMBER = ''
TWILIO_FROM_NUMBER = ''
```

References
----------

Twilio Sending Messages: https://www.twilio.com/docs/api/rest/sending-messages
